DarlingJS v0.0.3

Component & entity based javascript game engine. With dependency injections and modular architecture. Inspired by Ash, AngularJS and CraftyJS

Download this project as a .zip file Download this project as a tar.gz file

Current state is under active developing. All comments and feedback are welcome.

Release Notes

v0.0.3

Core
  • Entities, Components, Systems like in Ash by @richardlord
  • API like CraftyJS by @sorenbs
  • Injection and Modularity influenced by AngularJS by Misko Hevery
  • Covered by tests Jasmine, TravisCI, GruntJS
Pluggable Modules
  • Physics Emscripten Box2D 2.2.1 by Erin Catto and @kripken
  • Physics Box2D web 2.1a by Erin Catto and Ivan Kuckir
  • Renderer Pixi.js by @GoodBoyDigital
  • Map generator by Tiled map by @bjorn
  • Performance (FPS/Mem) metter Stats by @mrdoob
  • Basic Particle System
